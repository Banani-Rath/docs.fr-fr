# global (R&#233;f&#233;rence C#)
Le mot clé contextuel `global`, lorsqu'il vient avant l'[opérateur ::](../../../csharp/language-reference/operators/namespace-alias-qualifer.md), fait référence à l'espace de noms global, qui est l'espace de noms par défaut pour tout programme C\# et ne possède pas de nom.  Pour plus d'informations, consultez [Comment : utiliser l'alias d'espace de noms global](../../../csharp/programming-guide/namespaces/how-to-use-the-global-namespace-alias.md).
