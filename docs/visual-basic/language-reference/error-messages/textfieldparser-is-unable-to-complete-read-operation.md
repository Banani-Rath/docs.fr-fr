# TextFieldParser is unable to complete the read operation because maximum buffer size has been exceeded
L'opération ne peut pas être achevée car la taille maximale de la mémoire tampon \(10 000 000 octets\) a été dépassée.  
  
### Pour corriger cette erreur  
  
-   Assurez\-vous que le fichier ne contient pas de champs incorrects.
