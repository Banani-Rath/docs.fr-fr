# Error creating Win32 resources: &lt;error message&gt;
Le compilateur [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] appelle Assembly Linker \(Al.exe, également appelé Alink\) pour générer un assembly avec un manifeste. L'éditeur de liens a signalé une erreur pendant la création d'une ressource en mémoire. L'environnement peut poser problème ou votre ordinateur peut manquer de mémoire.  
  
 **ID d'erreur :** BC30136  
  
### Pour corriger cette erreur  
  
1.  Examinez le message d'erreur cité et consultez la rubrique [Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/fr-fr/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) pour obtenir davantage d'explications et de conseils.  
  
2.  Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
