# Les instructions &#39;Try&#39; ne sont pas valides dans la fen&#234;tre Ex&#233;cution
Les instructions `Try` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30717  
  
### Pour corriger cette erreur  
  
-   Supprimez les instructions `Try` du code de débogage.
