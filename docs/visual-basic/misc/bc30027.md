# &#39;End Function&#39; attendu
Une instruction `End` doit correspondre au bloc qu’elle termine. Une définition de procédure `Function` n’a pas été correctement terminée.  
  
 **ID d’erreur :** BC30027  
  
### Pour corriger cette erreur  
  
-   Ajoutez une instruction `End Function` à la fin de la procédure.
