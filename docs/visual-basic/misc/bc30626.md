# L’instruction &#39;Namespace&#39; doit se terminer par un &#39;End Namespace&#39; correspondant
Les blocs `Namespace` doivent se terminer par une instruction `End Namespace`.  
  
 **ID d’erreur :** BC30626  
  
### Pour corriger cette erreur  
  
-   Ajoutez une instruction `End Namespace` à la fin du bloc `Namespace`.
